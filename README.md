Escape: The Iron Haunt

Explore the shadows, survive the knights, and escape with the treasure.

Players Overview

Step into a high-stakes VR adventure where atmosphere is everything. Trapped within the walls of a haunted house, you must navigate torch-lit rooms to find a hidden treasure. But you aren't alone. Restless knights patrol the halls and ghosts lurk in the dark. Manage your health and find the secret exit before the house claims you forever. Do you have the courage to claim the treasure and make it out alive?

Proffesional Overview

The Iron Haunt is Built in Unity 2025 using the XR Interaction Toolkit, this project demonstrates advanced VR implementation including environmental triggers, Ui elements, and custom AI behaviors for an immersive horror experience.

How to Play
The primary platform for this game is SideQuest.

SideQuest (Recommended): Visit the game page on SideQuest and select "Sideload Now".

Itch.io: A standalone APK is also available on our Itch.io page for manual sideloading via the PC SideQuest app. Use the windows build to play on conformant OpenXR runtime (like SteamVR or the Oculus PC App). Make sure app is installed and set as your computer's default OpenXR runtime.

Controls

Thumbstick: Continuous Movement (Warning: May cause motion sickness).

Grip Button: Pick up items (Torch, Scythe, Treasure).

Trigger: Interact with UI elements.

Secondary Button: Open the Reset/Restart screen.

In-Game Instructions

OBJECTIVE: Find the treasure room, retrieve the treasure, and locate the hidden exit.

Survival: Use your scythe to defeat the Knights. Ghosts are spectral entitiesâ€”they cannot be killed, so stay alert for jumpscares!

The Torch: Use your light to navigate dark areas and trigger environmental events, such as burning through barriers to escape.

Difficulty: Select your challenge level at the start. "No Death Mode" (100 lives) is available for those who prefer exploration over challenge.

Game Over: If a knight defeats you, use the reset screen to try again.
